Proyectos de Evidencia

1. Sistema de Riego Creativo (Andravian) - IoT y Visualización de Datos

**Tecnologías:** **JavaScript (Fetch)**, **HTML/CSS**, **ThingSpeak API**, **Plotly.js**, **XLSX.js**, **PHP (para backend, si lo usaste)**.

Plataforma diseñada para el **monitoreo en tiempo real** de un sistema de riego. El proyecto incluye la captura de datos desde un dispositivo de hardware y una interfaz web robusta para la gestión y visualización de la información.

Evidencia de Ingeniería de Software (Lo que hice):

  • **Monitoreo en Tiempo Real (Consumo API):** Se implementó la lógica en JavaScript para consumir la **API REST de ThingSpeak** y aplicar un filtrado de datos por rangos de fecha definidos por el usuario, mostrando lecturas precisas de múltiples sensores.
  • **Manipulación y Exportación de Datos:** El sistema incluye la funcionalidad para exportar los datos filtrados directamente a formatos estándar de la industria como **Excel (.xlsx)** y **PDF** (usando librerías JavaScript como XLSX.js y jsPDF).
  • **Visualización Dinámica de Datos:** Se utilizó la librería **Plotly.js** para crear **múltiples gráficas dinámicas** que representan el historial de las lecturas de los sensores (Temperatura, Humedad, Lluvia, etc.), facilitando el análisis.
  • **Integración de Hardware/Software:** Gestión de la comunicación y el flujo de datos desde el dispositivo de hardware hasta la interfaz web.
